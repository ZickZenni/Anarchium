<p>

Anarchium is a Minecraft Mod that brings chaos to your world just like the
infamous <a href="https://www.gta5-mods.com/scripts/chaos-mod-v">Chaos Mod V</a>.

No additional content is getting added to your world, allowing you to remove it without any problems, but still giving a
new chaotic experience to play Minecraft.

**Note:** This mod is heavily work in progress and is still missing a LOT of features and effects.

</p>
